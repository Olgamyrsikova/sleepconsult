# Sleep Consult Web Page

## Run The Site

    $ bundle install
    $ bundle exec jekyll serve --watch

## Used Resources

* Jekyll https://jekyllrb.com
* Simple Icons https://simpleicons.org
* Slick Carousel https://kenwheeler.github.io/slick/
